> El modpack del servidor **NO** es para Forge, es para [Fabric](https://www.minecraft-hosting.pro/es/articulo/485-fabric), si instalas los mods en Forge, no funcionarán.
{: .prompt-danger}

## Mods 100% necesarios

El servidor **NO necesita** ningún mod (*0 mods necesarios*) para entrar y jugar normalmente, los mods mencionados en este post solo son necesarios en caso de que quieras:

- Mejorar tus FPS
- Usar el chat de voz
- Evitar la pantalla de carga del resourcepack
- Tener zoom
- Tener shaders (si tu PC lo aguanta)
- Tener minimapa

---

## Instalación automática

Para que no tengas que leer todo este post, hice un modpack que puedes instalar automáticamente en un launcher mucho mejor que el de Minecraft oficial.

El launcher recomendado es [Prism Launcher](https://prismlauncher.org), es fácil de instalar y de usar, con soporte para múltiples para cuentas no-premium una vez que se haya registrado por lo menos una cuenta premium oficial, por eso y más, se recomienda en este caso para poder instalar fácilmente el modpack del servidor.

Una vez que hayas instalado Prism, decide si quieres instalar la versión completa o la versión ligera de los mods...

### Versión completa

Elige esta versión solo si tienes una computadora suficientemente buena (en general los mods no son pesados... pero algunas computadoras no los aguantan) 

Esta versión incluye todos los mods mencionados en este post, además de un pack de shaders (solo para mostrar que puedes tener shaders, está deshabilitado por defecto).

<details>

### Versión ligera

Solo incluye los mods esenciales (para usar el chat de voz y para mejorar FPS) de este post, si tu computadora aguanta Minecraft, aguantará estos mods también.

### Instalación...

1. Teniendo Prism abierto, selecciona "Añadir instancia"![Paso 1, añadir instancia](step1.png)
2. "Añadir instancia" abre una ventana, en la que no tienes que cambiar nada, solo seleccionar "Importar"![Paso 2, seleccionar Importar](step2.png)
3. Al seleccionar importar, se te pedirá un enlace para descargar el modpack, los enlaces que puedes usaron son:
   - `https://link.buap.pro/modpackfull` - para la Versión Completa del Modpack
   - `https://link.buap.pro/modpacklite` - para la Versión Ligera del Modpack
![alt text](step3.png)
4. Pega o escribe el link del modpack que elegiste en el campo de importar...![alt text](step4.png)
5. Si quieres, puedes ponerle un nombre y grupo al modpack (es opcional, no cambia nada si lo haces o no), al tener el nombre que quieras, dale a "Ok" ![alt text](step5.png)
6. Al seleccionar "Ok" se empezará a descargar e instalar la nueva instancia con el Modpack ![alt text](step6.png)
7. Una vez que la instalación termine, aparecerá el modpack en tu lista de instancias de Prism ![alt text](step7.png)
8. Si todo sale bien, debería iniciar el juego y (posiblemente) abrirse una ventana de ejecución (consola) que tendrá un montón de texto, lo cual significa que seguramente todo salió bien... ![alt text](step8.png)

> Si el texto de la última ventana no se mueve por 5-10 segundos y no hay nada avisando sobre descargas, posiblemente algo salió mal...
{: .prompt-warning}
> Si tienes problemas con la instalación, puedes pedir ayuda en Discord o WhatsApp, seguramente alguien te pueda ayudar :)
{: .prompt-info}
> Usando la instalación automática, no tendrás que poner la IP, el modpack tiene el servidor añadido a tu lista de servidores ;D
{: .prompt-tip}
